# klok-bot

### 1. Clone Repository

```bash
git clone https://github.com/harunros66/klok.git
```

### 2. Go To Bot Directory
```
cd klok
```

### 3. Install
```
npm install axios https-proxy-agent socks-proxy-agent uuid
```

### 4. set up token bearer
```
nano tokens.txt
```

### 5. proxy

```bash
nano proxy.txt
```

### 6. run

```bash
node klok.js
```
